사용 방법
  1. 위 버튼 3개로 PNG 파일을 다운로드합니다.
  2. 다운로드된 파일을 C:\coding\rest01\ 폴더에 복사합니다.
  3. 터미널에서 아래 명령어를 실행합니다:
    git add og-image.png favicon.png apple-touch-icon.png
    git commit -m "add og-image and favicon" && git push
  4. 배포 후 카카오톡에 링크 공유 시 썸네일이 표시됩니다.
