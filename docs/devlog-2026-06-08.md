## 2. 상세 작업 내역

### 2-1. 라이트 모드 팔레트 적용 강화

**문제:** 팔레트를 Forest/Violet/Ocean/Slate로 변경해도 라이트 모드 컨텐츠 영역에서 색상 변화가 거의 없었음.

**원인:** `palettes.css`의 각 팔레트 블록에 `--surface`, `--card-bd`, `--shadow` 변수가 정의되지 않아 navy 기본값만 사용됨.

**해결:**
- `palettes.css`: 각 팔레트에 라이트 모드 서피스 색상 추가
  - Forest: `--surface: #f0fdf4`, `--card-bd: #bbf7d0`
  - Violet: `--surface: #faf5ff`, `--card-bd: #ddd6fe`
  - Ocean: `--surface: #ecfeff`, `--card-bd: #a5f3fc`
  - Slate: `--surface: #fff1f2`, `--card-bd: #fecdd3`
- `style.css`: `.info-card`에 팔레트 좌측 3px 액센트 바 (`border-left: 3px solid var(--p1-on-surface)`)
- `style.css`: `.sk-card`에 팔레트 상단 2px 액센트 라인 (`border-top: 2px solid rgba(var(--p1-rgb),.35)`)
- `style.css`: 다크 모드 `border-color` 오버라이드 이후 액센트 복원 추가

**결과:** 팔레트 전환 시 페이지 배경, 카드 테두리, 액센트 바가 모두 팔레트 색상으로 변경됨.

### 2-4. Imaging & Histology 카드 레이아웃 수정

**문제:** 스킬 카드(`.sk-card.wide`)의 텍스트가 데스크톱과 모바일 모두에서 좁게 뭉치는 현상.

**원인 분석:**

CSS Grid `grid-template-columns: 52px 1fr auto` 에 4개 자식 요소가 배치되면서:
```
[icon]  [h3]    [p text]       ← p가 auto 칼럼에 들어가 좁아짐
[sk-highlight]                 ← 다음 행으로 밀림
```

모바일에서 명시적 칼럼을 2개로 줄여도 `grid-column: 3` 으로 인해 암시적 3번째 칼럼이 생성되어 레이아웃 깨짐.

**해결:**

HTML: `h3`와 `p`를 `.sk-wide-body` wrapper로 묶어 **3개 자식**으로 단순화:
```html
<div class="sk-card wide">
  <div class="sk-icon">🔬</div>
  <div class="sk-wide-body">
    <h3>Imaging & Histology</h3>
    <p>IHC · IF · ICC · ...</p>
  </div>
  <div class="sk-highlight">Human & Mouse brain tissue analysis</div>
</div>
```

CSS: 3칸 그리드에 3개 자식이 1:1 대응:
```
Desktop: [icon 52px] [sk-wide-body 1fr] [sk-highlight auto]
Mobile:  [icon 44px] [sk-wide-body 1fr]
         [sk-highlight  ← grid-column: 1/-1 전체 너비]
```

**결과:** 데스크톱·모바일 모두에서 p 텍스트가 충분한 폭으로 렌더링됨.

### 2-5. 팔레트 전환 트랜지션

**목적:** 팔레트 버튼 클릭 시 색상이 즉시 바뀌어 어색했던 UX 개선.

**구현:**
```css
:root {
  transition: background-color 0.4s ease, color 0.4s ease,
              border-color 0.4s ease;
}
```
- `.sk-card`, `.info-card`, `body`, `nav` 등 주요 요소에 `transition` 추가
- 0.4s ease 타이밍으로 팔레트 전환 시 부드러운 크로스페이드 효과

**결과:** Navy → Forest → Violet 등 팔레트 전환이 0.4초 동안 부드럽게 애니메이션됨.
